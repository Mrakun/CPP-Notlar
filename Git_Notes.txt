# Genel Git Notları

## Başlangıç

- `git init` : Yeni bir git deposu başlatır.

- `git add <dosya>` : Dosyayı sahneye ekler (staging area).

- `git add .` : Tüm değişiklikleri sahneye ekler.

- `git status` : Çalışma alanının durumunu gösterir.

- `git restore` : Değişikliği geri alıp alanda tutmaya devam etmek için kullanılır.

- `git reset --hard` : Tüm değişiklikleri geri alır ve saklandığı alandan çıkartılır.

- `git remote add origin <url>` : Uzak depo ekler.

- `git remote -v` : Uzak depoları listeler.

- `git diff` : Değişiklikleri gösterir.

- `git diff --staged` : Geçici alandaki değişiklikleri gösterir.

- `git push -u origin <branch>` : Değişiklikleri uzak depoya gönderir.

- `git push` : Remote'a kayıtlı depoya gönderir

- `git pull` : Uzak depodan değişiklikleri çeker.

- `git clone <repo-url>` : Var olan bir depoyu klonlar.

- `git commit -m "mesaj"` : Sahnedeki değişiklikleri kaydeder.

- `git log` : Commit geçmişini gösterir.

- `git branch` : Mevcut dalları listeler.

- `git branch <yeni-branch>` : Yeni dal oluşturur.

- `git checkout <branch>` : Dal değiştirir.

- `git checkout -b <yeni-branch>` : Yeni dal oluşturup ona geçer.

- `git merge <branch>` : Belirtilen dalı mevcut dala birleştirir.

- `git stash` : Geçici olarak değişiklikleri saklar.

- `git rm <dosya>` : Dosyayı siler ve sahneye ekler.

- `git help <komut>` : Komut hakkında yardım alır.
